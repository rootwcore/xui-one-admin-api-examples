# Contributing

Thanks for contributing. This repository is a community reference for developers looking for XUI.ONE Admin API actions and examples.

## Contribution types

- Add missing API actions
- Add version-specific notes
- Add new cURL, PHP, Python, or JavaScript examples
- Improve documentation
- Fix typos or formatting issues
- Improve security guidance

## Pull request rules

1. Do not include real API keys, panel URLs, access codes, usernames, passwords, or customer data.
2. Use placeholders in every example.
3. Mention the XUI.ONE version where the action was tested, if known.
4. Include the expected result and any relevant error output when reporting behavior.
5. Keep examples simple and easy to copy.

## Placeholder format

```text
https://PANEL-DOMAIN:9000/ACCESSCODE/?api_key=YOUR_ADMIN_API_KEY&action=get_lines
```
